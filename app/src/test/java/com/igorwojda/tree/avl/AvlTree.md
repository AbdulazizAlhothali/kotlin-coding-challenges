# AVL

## Instructions

Implement [AVL tree](https://en.wikipedia.org/wiki/AVL_tree) data structure.

[Puzzle](AvlTree.kt) | [Solution](AvlTreeSolution.kt)

## Examples

```
// -------AVL tree-----------
//
//           9
//         /   \
//        7     6
//       / \   /
//      3   2 4
//
// --------------------------

val avl = AVLTree<Int>()
avl.add(9)
avl.add(7)
avl.add(6)
avl.add(3)
avl.add(2)
avl.add(4)
```
